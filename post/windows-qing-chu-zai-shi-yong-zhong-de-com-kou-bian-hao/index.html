Windows 清除在使用中的COM口编号

开始 -->> 在运行中输入：regedit  -->> 回车打开系统注册表  -->>  修改注册表对应的目录值如下：

HKEY_LOCAL_MACHINE
        -->> SYSTEM
                -->> CurrentControlSet
                        -->> Control
                                -->> COM Name Arbiter

打开COM Name Arbiter如下图

把ComDB的十六进制数据修改为0或者删除。

注：这样会重置所有COM编号，至于原先对应的的COM口驱动还是在的。

请到客户端“主题--自定义配置--valine”中填入ID和KEY
